A Hangman Game On Python  is a game in which the player is trying to guess a secret word. The player guesses letters, one at a time, and is told where each such letter appears in the secret word. If a guessed letter does not appear at all, it is considered a mistake. If the player makes certain no. of mistakes in total, the game is lost. It is a language game  and the  function  of  language games  is  to  practice  specific  language items  such  as  grammar,  vocabulary, and  spelling, and it is important to develop language skill.
